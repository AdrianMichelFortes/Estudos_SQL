# 📚 Estudos em SQL (Oracle SQL)

Durante meus estudos em SQL, desenvolvi uma base sólida em bancos de dados relacionais, aprendendo desde os conceitos fundamentais até a implementação de operações essenciais utilizadas no desenvolvimento de sistemas e na análise de dados.

Ao longo dos exercícios e projetos, pratiquei a criação e manipulação de bancos de dados utilizando **Oracle SQL**, trabalhando com comandos **DDL (Data Definition Language)**, **DML (Data Manipulation Language)** e consultas SQL para gerenciamento, organização e extração de informações.

---

# 🚀 Principais Conhecimentos Adquiridos

- ✔️ Criação e estruturação de tabelas (`CREATE TABLE`);
- ✔️ Definição de chaves primárias (**Primary Keys**) e chaves estrangeiras (**Foreign Keys**);
- ✔️ Modelagem de bancos de dados relacionais;
- ✔️ Inserção, atualização e remoção de registros (`INSERT`, `UPDATE` e `DELETE`);
- ✔️ Consultas para recuperação de informações utilizando `SELECT`;
- ✔️ Filtragem de dados com `WHERE`;
- ✔️ Ordenação de resultados com `ORDER BY`;
- ✔️ Criação e utilização de **Views** (`CREATE VIEW`);
- ✔️ Alteração da estrutura de tabelas (`ALTER TABLE`);
- ✔️ Exclusão de tabelas e objetos do banco (`DROP`);
- ✔️ Utilização de funções básicas para manipulação e análise de dados;
- ✔️ Compreensão dos relacionamentos entre tabelas (**1:1**, **1:N** e **N:N**);
- ✔️ Aplicação de boas práticas na organização de bancos de dados.

---

# 🎯 Objetivo

Este repositório reúne os exercícios e exemplos desenvolvidos durante meu processo de aprendizagem em SQL, demonstrando minha evolução na manipulação de bancos de dados relacionais e consolidando conhecimentos essenciais para aplicações em **Business Intelligence (BI)**, **Power BI**, análise de dados e desenvolvimento de software.

O principal objetivo é registrar minha evolução, praticar conceitos fundamentais de SQL e construir uma base sólida para projetos futuros envolvendo bancos de dados e análise de informações.
